What are HTTP verbs?

Normally when you access a website, your browser makes an HTTP request to GET content from the server. However, your browser isn’t always fetching data from a server, sometimes it POSTs data too (Such as when logging in to a website). HTTP verbs are the standardized method for telling the server what to do. The most common HTTP verbs are:

  • GET: fetch data from a server (eg: retrieve an HTML file)
  • HEAD: check what the server will do before making a GET request (eg: check response size before fetching something)
  • POST: post data to a server (eg: log in to a website)
  • PUT: update an existing resource
  • DELETE: delete a resource
